# Utility-Arrays
PHP arrays utilities.

Installing
---
Add following code to your "composer.json" file
`
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/php0id/Utility-Arrays"
        }
    ],
    "require": {
        "php0id/utility-arrays": "dev-develop"
    }
`
and run `composer update`
